> v-1.0 Desktop environment installer for Fedora Server Edition

### Features

* **Install** - Runs the installer for the Desktop Chooser.

### Getting Started

```shell
git clone https://github.com/Smiley-McSmiles/smileyLaptopSetup
cd smileyLaptopSetup
chmod ug+x Install
sudo ./Install
```
